水餃三溫暖　皮膚滑溜

最怕食到「皮開肉爛」的水餃，要煮到皮透明又滑嘟嘟，就要學Queenie的做法，煲滾後加凍水，重複三次，餃皮便透明又滑溜。

記者：黃潔蓮
攝影：伍慶泉

今日煮腦　12:30pm

陳莉敏（美顏食譜作者及美容專欄作家）

現任空姐、TVB8主持、美顏食譜作者、美容專欄作家和飾物設計師，曾出版《吃出美麗》，最愛的還是吃和煮。

菜式：
甜不辣

媽媽水餃

製作難度：★★★☆☆
時間：1小時

這個是Queenie媽媽的愛心水餃，每隻都脹卜卜，餡料多多，睇見都想食！

材料

韭菜/蝦/豬肉碎各1/2斤、水餃皮1斤

醃料：醬油2湯匙、香菇粉1湯匙、胡椒粉/麻油/糖各1茶匙、蛋1隻

調味料：胡椒粉少許、香菇粉1湯匙、鹽1茶匙

做法

1. 豬肉碎放大碗裏，加醃料撈勻，最好醃過夜，會更入味。

2. 韭菜洗淨瀝亁水，切粒備用。

3. 蝦挑腸，切粒，放雪櫃雪半小時，讓其吸亁水份。

4. 然後將韭菜粒、豬肉碎、蝦粒放大碗裏撈勻。再加鹽、香菇粉、胡椒粉調味，撈勻成餡料。

5. 用匙羮舀起兩茶匙餡，放水餃皮中間，在邊位搽少少水，然後對摺掐實條邊即可。

6. 大火煮滾一煲水，放入餃子，當水再次滾起，加少少凍水降溫，重複這動作3次。當水餃浮起，而又脹卜卜、皮呈半透明，可以撈起！食餃子時可配你喜歡的醬汁，豉油、辣椒醬即可。

貼士

水餃包好後可放冰箱儲存7日，若有剩餡料，可加蛋漿煎蛋角。

麻油薑葱炒麵

製作難度：★★★☆☆
時間：15分鐘

這個炒麵剛炒好不到3分鐘，已被攝製隊搶食一空，全因炒麵好煙韌，配上炒香了的蝦子，更香口呀！

材料

蝦子麵3餅、薑絲/葱段各4湯匙、麻油2湯匙、蝦子少許

調味料：蠔油1湯匙、醬油膏1/2湯匙、糖1/2茶匙

做法

1. 大火煮滾一煲水，放入蝦子麵煮1分鐘，然後撈起浸冰水，浸至麵凍後，撈起瀝亁水備用。

2. 燒熱鑊，加麻油，用中火爆香薑絲及葱段，然後加蝦子麵炒勻。

3. 加蠔油、醬油膏及糖調味，炒勻即可。

4. 將炒麵放入飯盒裏，加葱段、薑絲在上面，在旁邊加西蘭花，再撒上蝦子，即成！

貼士

想麵煙韌些，煮到一散開就要撈起。烚好的麵記得要浸冰水，會爽好多。

甜不辣

製作難度：★★★★☆
時間：30分鐘

台灣地道小吃甜不辣其實不甜也不辣，乃是取日文發音「Tempura」，即是天婦羅炸物之意。

材料

急凍魚肉350克、冰水100克、味椒鹽適量

調味料：鹽1茶匙、砂糖2湯匙、白胡椒粉1茶匙、地瓜粉80克

做法

1.魚肉與鹽放入攪拌機，冚蓋攪半分鐘至碎爛。

2. 將魚肉與白胡椒粉、砂糖、地瓜粉及冰水撈勻。用電動攪拌器以慢速攪拌5分鐘，至魚肉有黏性起膠。

3. 將魚肉漿放入密實袋，將其中一個角剪窿。中小火燒熱一鑊油，大概140℃，唧出魚漿，然後轉中火，如果油溫太低，炸出來會皺皮。

4. 當炸到部份甜不辣浮起，轉中小火，繼續炸到脹身，外表微微金黃色兼全部浮起，便可以撈起瀝亁油攤一攤。

5. 然後將油鑊轉大火，將甜不辣回鑊，將油逼出，即可上碟，撒上味椒鹽即成。

貼士

用來做甜不辣的魚最好選用白身魚，例如越南鯰魚；而三文魚、鱈魚因為太肥，較難起膠。

QQ朱古力

製作難度：★★☆☆☆
時間：1小時20分鐘

大大粒的朱古力，內有各式果仁及餅亁餡料，咬入口夠脆，用來送禮大方得體。

材料

黑朱古力300克、牛油50克、消化餅/杏仁/腰果各100克、提子亁50克、可可粉5湯匙、蜜糖2湯匙

做法

1. 將牛油放入不銹鋼碗，在熱水煲上細火慢慢坐溶。然後加朱古力，熄火，攪至朱古力溶掉就加蜜糖撈勻，備用。

2. 將消化餅搣碎成為細細塊。杏仁、腰果、提子亁切成細細粒。

3. 將提子亁、杏仁、腰果及消化餅放入大碗撈勻，然後加入朱古力溶液撈勻。

4. 在碟上鋪上錫紙，然後用匙羮挖一個朱古力波，放在錫紙上，放入雪櫃雪1小時至凝固，最後撒上可可粉即成。

貼士

材料配搭好隨意，但要留意朱古力跟果仁餡比例是1:1，即朱古力的重量，是果仁及餅亁重量總和。
