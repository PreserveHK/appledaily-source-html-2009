公園唱歌衝突　200人包圍執法者

【本報訊】移師屯門杯渡路（南）休憩公園繼續唱歌的曲藝團體，數月來不停以揚聲器跳舞唱歌，造成滋擾。警方昨午接到噪音投訴，一度進駐監視。一批阿伯與前來干涉的康文署人員發生爭執，氣氛緊張，逾200人一度與執法人員對峙，幸最後和平散去。康文署發言人稱已收到190宗噪音投訴，對事件關注。
現場的杯渡路（南）休憩公園，與屯門市鎮公園僅一河之隔。昔日有多個曲藝團體在屯門市鎮公園用揚聲器唱歌，多次與康文署人員衝突，至去年11月開始有曲藝團體成員移師杯渡路（南）休憩公園繼續唱歌。據稱由於幾乎每天有大群阿伯響應和打賞，曲藝團體成員索性長期在這地方唱歌，每天下午2時開始至晚上7時結束，平日吸引逾200名阿伯觀賞，周六和周日多至400名。

三個月接190宗投訴

附近有工廠大廈，不少商戶透露被揚聲器打擾，職員不能專心工作，連致電給顧客傾生意也因被音樂蓋住致聽不清電話，已多番投訴。昨午2時，兩男一女中年曲藝團體成員如常拖來電子琴、色士風和咪到場開工演唱，康文署人員干涉無果。
至下午4時警方收到噪音投訴，遂派出數名軍裝警員到場調查和監視。最高峯時聚集200多名阿伯聽歌。康文署人員再次出面勸他們勿用揚聲器，遭幾十名阿伯用粗言穢語挑釁。有阿伯說：「食狗屎喇！」康文署人員回應說：「唔好講呢啲廢話喇！」場面頗為火爆，康文署人員向警員報稱遭推撞，警員上前調查。目擊者稱沒人動手，警方調查後無拘捕任何人。及至晚上7時，曲藝團體成員和觀眾亦陸續散去。
康文署發言人稱，其實由去年11月至今年1月底，該署共收到190宗該公園的噪音投訴，最近已進行數次執法行動，向出動揚聲器和使用淫褻性語言的多名違規市民索取個人資料，以便進行檢控，今後將連同警方和環保署人員加強執法。
