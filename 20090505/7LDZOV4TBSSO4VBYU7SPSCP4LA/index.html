沙律炸魚柳

0hr20min
製作難度：★★☆☆☆

炸魚柳一般都會用急凍的龍脷便可，因為新鮮龍脷柳很難買得到。

材料

龍脷魚柳1條、蘋果1個（去皮切細粒）、啤梨1個（去皮切細粒）、士多啤梨3粒（切粒）、沙律醬6湯匙、雞蛋1隻（拂勻）、麪粉18克、麪包糠75克、水2湯匙

醃料：鹽/雞粉各1/3茶匙

做法

1. 將所有生果拌勻，撈上沙律醬備用。

2. 將魚柳加入鹽及雞粉塗抹兩面備用。

3. 將蛋漿、水及麪粉倒入大碗拌勻，塗上魚柳兩面，再均勻蘸上麪包糠備用。

4. 燒滾油約70℃，放入魚柳用慢火炸3分鐘，炸至金黃色即上碟，伴上生果沙律即成。

貼士

炸魚柳時記得要用慢火，一開大火就好易將麪包糠炸燶。

秘製沙嗲牛肉包

0hr45min
製作難度：★★☆☆☆

煮沙嗲牛肉包的汁一定要埋少許生粉芡，可令牛肉更滑身，而且沙嗲汁又不會煮得太稀，沾濕了麪包就不好吃了。

材料

豬仔麪包2個、牛肉片56克、西生菜絲38克、牛油1湯匙

醃料：雞粉1/3茶匙、梳打粉/生抽/生粉各少許

醬料：沙嗲醬1湯匙、生粉水少許、紅椒粒1茶匙、洋葱粒1茶匙、蒜茸1/2茶匙、糖1/3茶匙

做法

1. 牛肉片加醃料拌勻醃30分鐘。

2. 將麪包切開，塗上牛油，開大火燒熱易潔鑊，把麪包塗有牛油的一面先放落鑊煎香，再輕烘包面，取出備用。

3. 燒滾油約70℃，將牛肉拉油取出備用。

4. 燒熱鑊下油，加蒜茸、紅椒粒、洋葱粒及沙嗲醬炒勻，再下糖，將牛肉回鑊，下生粉水埋芡煮至稠身盛起。

5. 將西生菜絲夾在麪包中，鋪上沙嗲牛肉即可。

貼士

麪包不宜煎過耐，否則會變乾身不夠鬆軟。

黑椒豬扒炒一丁

1hr15min
製作難度：★★☆☆☆

炒即食麪要炒得爽口彈牙才好吃，秘訣是泡麪時不要泡太耐，不然炒出來的麪就會過腍。

材料

豬扒2件、即食麪2包、黑椒粒1/3茶匙、洋葱絲1/2個、椰菜絲/芽菜各56克、西芹絲/紅椒絲/葱段各18克、即食麪調味粉1包、麻油/老抽各1/2湯匙

醃料：梳打粉/糖/雞粉各1/2茶匙、生抽1/2湯匙、生粉少許

做法

1. 豬扒加醃料拌勻醃1小時備用。

2. 即食麪用滾水泡1分鐘至散開，撈起瀝乾水備用。

3. 燒滾油約70℃，將豬扒放入炸3分鐘至熟，撈起切條備用。

4. 燒熱鑊下油，放洋葱絲、芽菜、西芹絲、紅椒絲、椰菜絲及葱段炒香，下即食麪炒勻，然後加入黑椒粒、老抽、即食麪調味粉及麻油炒香，將豬扒回鑊炒勻即可上碟。

貼士

加入黑椒粒去炒麪，會更加惹味。

蜜糖雞翼拼薯條

0hr45min
製作難度：★★☆☆☆

陳師傅炸薯條喜歡用大隻白皮薯，切出來長身些，質感較鬆化；其他如紫皮薯仔，口感較實，適合做焗薯。

材料

雞翼12隻、薯仔1個、蜜糖5湯匙、芝麻/鹽/水各少許、麪粉2湯匙、生抽2茶匙

醃料：鹽1/3茶匙、雞粉/咖喱粉各1/2茶匙、生抽1/2湯匙、生粉/麪粉各1湯匙

做法

1. 雞翼加醃料撈勻醃30分鐘備用。

2. 薯仔去皮切條，略冲水後瀝乾備用。

3. 燒滾油約70℃，放雞翼後收細火炸5分鐘後轉大火炸，見外皮炸至金黃色可撈起。

4. 將瀝乾薯條撒上鹽及麪粉撈勻，開大火燒滾油至80℃，放入炸2分鐘至金黃撈起。

5. 開大火燒熱鑊，加蜜糖、水及生抽煮勻，將雞翼回鑊兜勻上碟，撒上芝麻，薯條伴碟。

貼士

薯仔切條後冲水，可冲走澱粉質，會較易上粉。
