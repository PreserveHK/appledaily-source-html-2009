冰凉鮮果冷麪（2人份量）

0hr20min
製作難度：★★★☆☆

天氣太熱不想吃肉，多吃水果就能解渴消暑，這個冷麪用鮮果配搭，味道清新又易做。

材料

蕎麥麪100克、西瓜3片、菠蘿3片、冰水1大碗、葱粒2湯匙

調味料：日本豉油2湯匙、日本芥辣適量

做法

1.將蕎麥麪放入滾水中，用大火煮3至4分鐘，盛起即放入冰水中浸泡。

2. 西瓜切塊；菠蘿切塊。

3. 將日本豉油與芥辣混成醬汁，加入葱粒攪勻，放入冰粒，吃時就會特別冰凉。

4. 然後將蕎麥麪瀝乾水放上碟，旁邊放生果，吃時記得蘸醬汁呀！

貼士

市面上買到多是支裝如牙膏狀的芥辣，另一種是以青芥辣粉開水調勻而成，兩者都可以用。但如買到新鮮青芥辣，即磨即吃，效果當然最好啦！

果香咖喱蝦（4人份量）

0hr30min
製作難度：★★★☆☆

印度人是靠吃咖喱來發汗消暑，怕辣的可學Angel加鮮果來煮，辣中帶點甜味，特別醒胃。

材料

中蝦仁6両、芒果1個、蜜瓜200克、乾葱茸3湯匙、咖喱粉2茶匙、椰漿1/3杯、生粉少許、水1/2杯、葡萄籽油2湯匙

調味料：鹽3/4茶匙、糖1/3茶匙、有機黑豆豉油1茶匙

做法

1. 將芒果切半去核，去皮後切塊；蜜瓜切塊。

2. 蝦仁洗淨抹乾，加生粉撈勻。

3. 用中小火燒熱鑊加葡萄籽油，爆香乾葱茸及咖喱粉，然後加水、鹽、有機黑豆豉油、糖及椰漿煮成咖喱汁。

4.然後放入蝦仁煮至滾起，加芒果及蜜瓜炒勻後即可上碟。

貼士

煮生果不宜用猛火煮及煮太耐，放入生果後炒勻就要熄火，否則芒果會煮得爛溶溶。如怕炒得不夠時間，生果不夠熱騰騰，可預先用熱水浸芒果3至5分鐘，浸至熱暖才炒即可。

三色牛蒡絲（4人份量）

0hr20min
製作難度：★★☆☆☆

除用辣味刺激味蕾，賣相亦可刺激食慾，這個三色牛蒡絲，紅黃綠三色好醒神，可引起食慾。

材料

牛蒡1/3條、小青瓜2條、黃椒/紅椒各1/2個、粉絲1札、乾九層塔適量、白醋2湯匙

調味料：麻油/生抽各2湯匙、糖2茶匙、黑胡椒/鹽各適量

做法

1. 牛蒡洗淨後切絲，浸入混了白醋的開水中，以防變黑。

2. 粉絲浸軟後瀝乾水後切碎；小青瓜去蒂切絲；紅椒及黃椒去蒂去籽後切絲。

3. 在鍋中加入適量水，加青瓜絲、黃紅椒絲、粉絲及牛蒡絲灼3分鐘至熟，盛起瀝乾水。

4. 將麻油、生抽、糖、黑胡椒及鹽撈勻，然後淋上（3），撈勻後上碟，撒上乾九層塔即成。

貼士

這個菜可變成凉拌菜，把灼熟的蔬菜絲立即放入冰水中浸5分鐘，盛起瀝乾再加調味料即可，會更爽脆呢！

麻辣雞絲（4人份量）

0hr15min
製作難度：★★☆☆☆

雞絲與芋絲吸收了麻辣汁，可以令胃口大增。

材料

雞胸肉1件、芋絲1盒、米酒1湯匙

麻辣醬料：葱1棵（切碎）、蒜末3湯匙、辣椒1隻（去籽切碎）、有機黑豆豉油5湯匙、冷開水1/2杯、芝麻醬6湯匙、糖3湯匙、醋2湯匙、花椒粉1茶匙、辣椒油2至4湯匙

做法

1. 開大火煮滾半煲水，加米酒、雞胸肉及芋絲，見雞胸肉表面變白色，撈起切開三件，再放入煲中，當煮至滾起後轉中火，前後合共約煮3至5分鐘，當雞胸肉煮熟後熄火盛起。

2. 雞胸肉稍放凉後，斜切成絲。

3. 芝麻醬加冷開水調稀，加有機黑豆豉油、糖、醋、花椒粉及辣椒油拌勻，再加葱粒、蒜末及辣椒碎拌勻即成為醬汁。

4. 在碟上先放芋絲，再鋪雞絲，將麻辣醬汁淋在雞絲上，撒上少許葱花即成。

貼士

切雞胸肉應逆紋切，如果順紋切，雞肉會難以咬爛，覺得好韌。
