慢慢煮の秘蜜

女人的皮膚是最好的濕度探測器，天氣稍乾，細紋、乾皮、龜裂都會瞬即出現。而最佳的天然潤膚品，非蜜糖莫屬。中國人講究全面護理，外敷之餘亦要內服，用蜜糖冲水飲嫌太普通，可以跟輝哥用四種不同的蜜糖來煮餸，他教路：「煮蜜糖一定要用慢火呀！」

學煮蜜秘訣　送你自助晚餐券

煮餸多數用砂糖，為何今次輝哥會用蜜糖呢？「砂糖多是加工精製的，當然不及蜜糖天然和健康，而且蜜糖的味道選擇很多，亦可自己加配料去煮製特色蜜糖，例如我用了新鮮人參去熬製人參蜜等，香甜而且有潤喉清熱的效果。」
至於每種蜜糖，也有其獨特之處。

野蜂蜜：由野蜂採集的蜜糖，經風吹日曬變成結晶，所以又稱石蜜，具有潤肺止咳、化痰及治胃病等功效。

西洋菜蜜：色澤帶綠，亦較稀，適用於冲水飲，如要烹調至稠，需以細火慢煮。

人參蜜：人參會有微苦味，與蜜糖調和就可中和苦味，要慢火熬製帶出人參清香。

龍眼蜜：味道甜且有濃濃的龍眼香味，呈深琥珀色，適合用來做甜品。

炸鯛魚柳配　野蜂蜜甜酸汁

0hr15min
製作難度：★★★☆☆

一般做炸漿要令它鬆脆，可以加泡打粉，不過輝哥用了傳統方法，用啤酒與暖水來溝麪粉，一樣可令粉漿酥脆。

材料

鯛魚柳150克、鹽/黑胡椒/麪粉各少許

炸漿料：啤酒150毫升、蛋1隻、麪粉11/2杯、芝士粉20克、暖水1茶匙

甜酸汁料：胡椒粉/蒜茸/鹽各少許、青檸1/2個（榨汁）、白醋1茶匙、乾葱1/2粒（剁碎）、野蜂蜜/生粉水各適量

做法

1. 將炸漿料撈勻。

2. 鯛魚柳切件，加鹽及黑胡椒調味，再撲上麪粉。

3. 燒熱一鍋油，將魚柳蘸上炸漿，放滾油中，先以大火將表面炸至硬身，然後轉中火炸分半鐘，瀝乾油上碟。

4. 將野蜂蜜、白醋及少許水落鑊，煮至野蜂蜜溶掉，再加青檸汁、胡椒粉及鹽調味。另起鑊炒香蒜茸及乾葱粒，然後放入汁煮熱，埋生粉水芡，配炸魚齊食。

貼士

市面上亦有些現成的炸漿粉，只要加水和油，便可使用。

烤牛肋條配　人參蜜糖汁

0hr45min
製作難度：★★★☆☆

牛肋條肥瘦均勻，肉酥軟，用人參味的蜜糖去煮，可辟去牛肉的羶味，亦可添香。

材料

牛肋條400克

醃料：紅辣椒粉1/2湯匙、黑胡椒碎/香蒜粉各3湯匙、幼鹽3/4茶匙

人參蜜糖汁料：鮮人參/蜜糖/洋葱各80克、燒汁150毫升、橄欖油2湯匙

做法

1. 將醃料撈勻。牛肋條以三條穿成一串，撒上醃料略醃。

2. 燒熱坑紋鑊，放入牛肋條，煎至兩邊金黃，然後放入焗爐以180℃焗30分鐘，烘至全熟。

3. 將洋葱及鮮人參切成碎粒後放入鑊，加橄欖油炒至金黃，然後加蜜糖及燒汁煮2分鐘成汁料。將汁倒上碟，放上牛肋條即成。

貼士

牛肋條用竹籤串起，可防止在烘焗時捲曲變形。

烤燈籠椒沙律伴　藍芝士西洋菜蜜汁

0hr20min
製作難度：★☆☆☆☆

沙律汁一向感覺偏酸，但加了西洋菜蜜調味，中和了藍芝士的鹹味及酸忌廉的酸味，汁濃而酸甜醒胃。

材料

青椒/紅椒/黃椒/紅洋葱各1/2個、雜香草5克、蒜頭1粒（剁碎）、藍芝士/酸忌廉各20克、西洋菜蜜/橄欖油各20毫升、乾葱碎1湯匙、黑胡椒碎/鹽各少許、雞毛菜適量

做法

1. 半份橄欖油、西洋菜蜜、藍芝士、酸忌廉及乾葱拌勻成沙律醬。

2. 將青椒、紅椒、黃椒、剩餘橄欖油、鹽、黑胡椒碎、雜香草及蒜茸拌勻醃一醃，放坑紋鍋上煎熟，然後切角片。

3. 三色椒放大碗中，加紅洋葱及半份沙律醬撈勻。在碟上放雞毛菜，放上三色椒，再淋上剩餘的沙律醬即成。

貼士

將燈籠椒先烤再撈沙律醬，除了更香，亦可避免出水，令沙律醬變得水滴滴。

龍眼花蜜　芝士蛋糕

1hr15min
製作難度：★★★☆☆

龍眼蜜味甜，配芝士蛋糕的香軟及清甜果味，減低芝士膩滯感。

材料

忌廉芝士100克、龍眼花蜜60克、糖60克、蛋1隻、淡忌廉150克、滾水50毫升、魚膠粉10克、直徑三吋蛋糕片2片、龍眼/藍莓各適量

做法

1. 將魚膠粉與10克糖加滾水拌勻坐溶。

2. 忌廉芝士、龍眼花蜜、糖、蛋及（1）拌勻。淡忌廉打拂至七成，然後拌入芝士漿中。

3. 將一片蛋糕放入模內，注入半份芝士漿，再放入另一片蛋糕，再注入剩餘芝士漿，然後放入雪櫃雪1小時至凝固。

4. 用火槍燒燒模四周，然後脫模，加上龍眼及藍莓作裝飾即成。

貼士

忌廉打拂至七成，即是打至拉起呈軟峯狀，如打拂至十成，蛋糕便會太硬。
