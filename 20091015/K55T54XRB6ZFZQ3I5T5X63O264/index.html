香煎大海蝦配鮮茄汁

0hr45min
製作難度：★★☆☆☆
份量：4人

材料

大老虎蝦4隻、番茄4個（切件）、清水800毫升、葡萄芥花籽油6湯匙

調味料：鹽適量、糖適量

做法

1. 落6湯匙油落煲，加番茄及水，用中火大約煮二十分鐘，煮到番茄熟爛，番茄連湯過篩去皮及籽。

2. 將番茄醬放回煲用中火再滾，等番茄醬轉稠身撈起。

3. 燒熱油鑊，將已洗淨的大老虎蝦，用大火煎至兩面金黃色。

4. 將番茄醬倒入至湯汁收濃，再加糖、鹽調味即可。

貼士

整呢個菜時，我會用多一點油，因為煎大蝦時可以更加香口，大家別以為食油唔好，營養師話，每人每日攝取5-6茶匙健康食油，對身體有益。

百合牛肉粒

0hr30min
製作難度：★★☆☆☆
份量：4人

材料

牛柳粒250克、蜜糖豆12條、鮮百合1-2個、蒜頭1個（去皮切片）、XO醬1湯匙、葡萄芥花籽油2湯匙

調味料：糖半茶匙、生粉半茶匙、生抽2茶匙、胡椒粉適量、麻油2茶匙、生粉水適量

做法

1. 將牛柳粒加入糖、生粉、生抽、胡椒粉、麻油，醃約10分鐘。將鮮百合分開，用清水清洗。

2. 燒熱鑊，落2湯匙油，大火爆香蒜片，加入牛柳炒到八成熟。

3. 再加入蜜糖豆、百合，最後加入XO醬快手拌均，最後加生粉水埋芡即可。

貼士

炒牛肉時切忌炒得太耐，如果唔係就會太韌，所以記住要用大火同快炒至好。

燴獅子頭釀栗子

0hr45min
製作難度：★★☆☆☆
份量：4人

材料

免治豬肉（後蹄肉）300克、薑茸2茶匙、已灼熟去殼栗子6粒、小棠菜適量、甘笋適量、罐裝笋適量、葡萄芥花籽油4湯匙

肉丸調味：生抽2湯匙、水1-2湯匙、砂糖2茶匙、粟粉1茶匙、鹽少許、麻油2茶匙

芡汁：生抽1湯匙、老抽半湯匙、砂糖2茶匙、粟粉2茶匙、水250毫升

做法

1. 將免治豬肉碎加入生抽、水、砂糖、粟粉、鹽、麻油同薑茸，向同一方向攪拌至膠狀。

2. 用大約1湯匙份量的豬肉，將栗子釀入，搓成丸狀，製成6粒肉丸。

3. 燒熱油鑊，放入肉丸，用中火煎至金黃色及變硬身，瀝乾油。

4. 預備小瓦煲，燒熱油，用大火爆香薑茸，甘笋及笋，再落小棠菜炒至軟身，加入肉丸，下芡汁轉慢火燴煮10分鐘，整煲端上即成。

貼士

以前買回來的栗子，要剝殼才可以用，現在超市可以買到剝了殼的栗子，方便得多。

炸釀大蘑菇

0hr30min
製作難度：★★☆☆☆
份量：4人

材料

荷蘭大蘑菇2隻、免治豬肉200克、乾冬菇4隻、雞蛋1隻、葡萄芥花籽油120毫升

調味料：生粉2茶匙、砂糖適量、鹽適量，生粉適量（撲蘑菇用）

做法

1. 將乾冬菇浸至軟身，瀝乾水切粒。

2. 將免治豬肉、加入冬菇粒、雞蛋、生粉、砂糖、鹽放入攪拌直至成膠狀。

3. 大蘑菇去蒂，將免治豬肉釀入大蘑菇上，兩邊撲上生粉。

4. 燒熱油鑊，放入大蘑菇，先將釀有肉的一面，以中火煎炸3分鐘，再炸另一面2分鐘即成。

貼士

豬肉內加入冬菇粒，除了可以增加菇的香味，還可以加強口感及味覺層次。

葡萄籽油抗氧化

香港人注重健康，尤其喜歡選用健康食油，藉以預防心血管疾病、糖尿病、肥胖症、以至抗衰老。一般含低飽和脂肪酸，不含膽固醇和反式脂肪，但同時含高不飽和脂肪酸，包括多元及單元不飽和脂肪酸的食油，正是健康之選。將葡萄籽油及芥花籽油混合，能提供身體適當的脂肪平衡，有助改善血脂，提升心臟健康，當中的維他命E和原花青素等抗氧化物，更有助抗衰老。
