「拍佢啲戲實死得」
吳彥祖自爆好友辱楊凡

由楊凡自編自導的電影《淚王子》，前晚在銅鑼灣舉行首映禮，演員關穎和朱璇齊以性感露肩裙示人。專程捧導演場的吳彥祖，指楊凡是其恩師，又自爆當年拍《美少年之戀》時，曾收好友短訊指楊凡專拍三級片，「拍佢啲戲實死得」。

代表香港出戰今屆奧斯卡金像獎最佳外語片的《淚王子》，前晚在銅鑼灣舉行首映禮，導演楊凡、演員關穎、朱璇、張孝全和范植偉等均有到場，並獲吳彥祖、林嘉欣、at17、林子祥和鄭佩佩等畀面撐場。

「第一場就錫」

穿上性感低胸晚裝示人的台灣女星關穎，前晚與朱璇如孖公仔般合照，兩人更表現得甚有默契，原來兩人在片中有一場接吻戲，關穎坦言拍攝當日十分尷尬，她說：「因為第一場就錫，而且是我做主動，好像想侵略對方一樣，我給女生的螢幕初吻就給了朱璇，當時我真的很緊張，拍完之後只是跟對方講不好意思，始終大家不熟，錫女生感覺怪怪的。」
專誠往捧導演場的吳彥祖則坦言視楊凡為恩師，因為他出道第一齣戲《美少年之戀》就是楊凡執導，他更自爆一件搞笑事說：「嗰時有朋友send咗中文sms畀我，我唔識睇，所以叫導演幫我睇，點知個sms係話楊凡啲戲係三級片，拍佢啲戲實死得，其實佢係藝術導演，我好欣賞佢。」
吳彥祖又笑言當初剛入行甚麼也不懂，幸導演每天抽空教他做戲，才造就今日的吳彥祖，他更謂自己當年好窮，連食飯都冇錢，所以導演每天都會吩咐工人煮飯給他食，令他十分感動。

林嘉欣願續約

此外，林嘉欣表示認識楊凡多年，提到楊凡作品的女主角多性感戲份，林嘉欣坦言不介意性感，最重要是導演和演員互相信任。近日有傳她會離開星皓，她表示與公司還有合約，但是否續約要看星皓的誠意，她說：「都有好多人接觸過我，但我會同星皓傾咗先，亦冇接觸金牌大風。」
