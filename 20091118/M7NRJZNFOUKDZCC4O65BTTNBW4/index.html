疑雨雪侵害　山西塌山泥
5戶民工23口全部罹難

山西發生山泥傾瀉事故，被活埋的23人，昨日全部死亡，死者為來自雲南的五個民工家庭，其中男性12名，女性11名。遇難者中有8名兒童，最大的4歲，最小的僅1歲。當局稱，事故初步疑與近日嚴重的雨雪災害有關，但具體原因仍在調查中。網民悲痛之餘，要求當局「不要總把責任歸咎於自然災害……該好好檢討了。」

搶救人員經近16小時挖掘，昨日凌晨，在呂梁市中陽縣張子山鄉張家咀茅火梁山泥傾瀉現場，共挖出23具屍體。他們為了到當地的煤礦工作，舉家搬遷，居住在事故現場臨時房屋，不料客死異鄉。
「就聽『嘩』一聲，我們全家就被泥漿衝到床底下了。」事故中幸運逃生的民工王榮策，昨懷裏抱着剛滿51天的小女兒講述事發過程時，仍猶有餘悸。他20歲的妻子徐霞也被嚇壞，不時抽泣。
來自陝西的王榮策稱，前日上午10時40分發生山泥傾瀉時，他和妻子、女兒還在屋內睡覺，但瞬間被驚醒，「災難來得太快了，整個床被打翻了。」他的家因沒有倒塌才逃過一劫。但三人全身是泥，「女兒被拽出來時簡直就是一個泥孩，都不會哭了。」據現場人士初步分析，山泥傾瀉疑與近日嚴重的雨雪災害有關係，因冰雪融化造成了山泥傾瀉，但當局未確定事故具體原因。

廣州罕見持續濕冷

此外，中國南方大部份地區及西北地方的雨雪天氣將持續到20日。廣州今日將降至6℃。氣象專家表示，廣州出現了30年來罕見11月中旬持續濕冷的情況。受暴雪影響，湖北和安徽等災情嚴重，多處地方出現天然氣短缺，部份企業被暫停天然氣供應，以保證民居需要。
